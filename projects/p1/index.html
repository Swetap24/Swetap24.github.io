Aerial Projection of 3D Hologram Object



In this project, I presented an interactive aerial projection of 3D hologram objects by using the pyramid hologram and parabolic mirror system (for 3D hologram object reconstruction). This system was designed to reconstruct and project the 3D hologram object in the mid-air. There are two main steps: the reconstruction of 3D object, and the projection of 3D hologram object in the mid-air. The first step is realized by using pyramid hologram with LCD display. The second step is achieved with the parabolic mirror hologram.

This project approaches how precisely the three dimensional images are framed and how the hologram functions. The project portrays two systems to extend the 3D multi-dimensional image aerially: First being the computer generated hologram(CGH) and the other spatial light modulator(SLM) method which embrace pyramid 3D image hologram technique to reconstruct the 1D image and concave lens combination to launch it in mid-air.
